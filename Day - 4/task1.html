FRONT END DEVELOPMENT

HTML

HTML stands for HyperText Markup Language and is the foundational language for creating web pages and structuring web content.It uses "tags" to define elements like headings, paragraphs, links, and images, whichbrowsers interpret to display the content of a webpage. HTML provides the basic structure, which is then enhanced with CSS for styling and JavaScript for interactivity,forming the backboneof the modern web


CSS

CSS,which stands for Cascading Style Sheets, is a style sheet language used in web development to describe the presentation and formatting of an HTML (or XML) document.


Javascript

JavaScript is a high-level, interpreted programming language primarily known for enablinginteractive and dynamic content on web pages. It is one of the three core technologiesof the World Wide Web, alongside HTML (which provides structure) CSS and (which handles styling).
